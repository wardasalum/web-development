INTRODUCTION



What is C#? C# is pronounced "C-Sharp". It is an object-oriented programming language created by Microsoft that runs on the .NET Framework. C# has roots from the C family, and the language is close to other popular languages like C++ and Java. The first version was released in year 2002. The latest version, C# 11, was released in November 2022. C# is used for:

  • mobile application
  • desktop application
  • website
